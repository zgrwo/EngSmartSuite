# SmartExcel Suite

> 工艺数据分析工具箱 —— 在 Excel 中选择数据，点击按钮，得到分析结果和报告。无需离开 Excel，无需编写代码。

## 这是什么？

SmartExcel Suite 是面向制造工艺工程师的 Python+Excel 数据分析插件。它把 Python 生态强大的统计分析能力（pandas、scipy、statsmodels、scikit-learn）封装为 Excel 功能区按钮，让你用熟悉的 Excel 界面完成从数据到结论的全流程。

**你不再需要：**
- 在 Minitab 和 Excel 之间来回搬运数据
- 每次分析都写 Python 脚本
- 把结果手动复制粘贴到 PPT 汇报材料

**你现在可以：** 选中数据 → 点按钮 → 得到分析结论 + 图表 + 报告（Excel/PDF/PPT）。

## 什么时候使用？

### 场景一：出了问题，要找原因（要因分析）

> "这周的不良率突然升高了，是料温的问题？模温？还是注射压力？"

| 分析方法 | 它能回答什么 | 什么时候用 |
|---------|------------|-----------|
| **相关性分析** | 哪些工艺参数与不良率/强度最相关 | 刚拿到数据，快速扫描所有因子的影响力 |
| **ANOVA** | 原料类型/机台/操作工 是否对质量有显著影响 | 判断某个类别因子（如材料牌号）是不是关键 |
| **假设检验** | 新旧工艺有没有"真的"差异 | 工艺变更前后对比、设备 A vs B 对比 |
| **决策树** | 如果多个参数都在变，哪个才是真正的根源 | 因子多、交互复杂、相关性分析不够直观时 |
| **VIF 共线性诊断** | 你的参数之间是否互相纠缠 | 建模之前检查"假相关"（两个参数实际是同一个东西） |

### 场景二：要找最优参数（DOE / 优化）

> "温度和压力调到多少，强度最高、不良率最低、成本还能接受？"

| 分析方法 | 它能回答什么 | 什么时候用 |
|---------|------------|-----------|
| **回归建模** | 建立 Y = f(X₁, X₂, …) 的数学公式 | 需要量化"参数变 1 单位，结果变多少" |
| **响应面分析** | 生成 3D 曲面图，可视化最优区域 | 有两个关键参数需要同时优化时（如温度+压力） |
| **网格搜索** | 在参数范围内自动找到最优组合 | 已经有回归模型，需要精确的最优值 |
| **多目标优化** | 同时优化强度和成本（一个升一个降怎么办） | 质量、成本、效率三者需要权衡时 |
| **DOE 效应分析** | 全因子/部分因子实验中，哪些效应最强 | 分析正式 DOE 实验矩阵的数据 |

### 场景三：要持续监控（过程控制）

> "产线是不是稳定的？会不会快要出问题了？"

| 分析方法 | 它能回答什么 | 什么时候用 |
|---------|------------|-----------|
| **SPC 控制图** | 过程是否受控，有没有异常点 | 日检/批检数据的日常监控 |
| **过程能力 Cp/Cpk** | 你的工艺是否能稳定满足规格要求 (Cp/Cpk ≥ 1.33?) | 客户审核、质量体系认证、新模具验收 |
| **趋势预测** | 接下来几天参数会往哪个方向漂 | 参数逐渐偏离目标值，需要提前干预 |
| **异常检测** | 这一批数据里有没有离群点 | 大量数据中快速找出"不对劲"的那几行 |

## 怎么安装？

### 前提条件

- Windows（macOS 也可运行，但 Excel 集成功能仅 Windows 完整支持）
- Python 3.10 或更高版本
- Microsoft Excel

### 安装步骤

```bash
# 1. 安装 SmartExcel
pip install smartexcel

# 2. 安装 Excel 加载项（这一步让 Excel 里出现"工艺分析"功能区）
xlwings addin install
```

### 验证安装

打开 Excel，你应该在功能区看到 **"工艺分析"** 标签页。如果没有，运行：

```bash
xlwings addin install
```

然后在 Excel 中：文件 → 选项 → 加载项 → 管理 Excel 加载项 → 勾选 xlwings。

## 怎么用？

### 方式一：Excel 功能区（推荐，零代码）

1. **打开你的数据文件** —— Excel 工作表，第一行为表头

2. **点击功能区 "工艺分析" 标签**

3. **选择分析方法**：
   - **要因分析** → 相关性分析 / ANOVA / 假设检验
   - **DOE/优化** → 回归建模 / 响应面分析 / 最优搜索
   - **过程监控** → SPC 控制图 / 过程能力

4. **在弹出的输入框中**：
   - 输入**目标列名** (Y) —— 你想解释/优化的指标，如 "不良率"
   - 输入**因子列名** (X)，逗号分隔 —— 可能的工艺参数，如 "料温, 模温, 注射压力"

5. **查看结果** —— 分析结果自动写入新的工作表，包含：
   - 分析结论（一句话，工艺语言）
   - 数据表格（如 ANOVA 表、相关系数矩阵）
   - 统计图表（如控制图、响应面 3D 图）

6. **导出报告** —— 点击"报告输出"区域的按钮，一键生成 PPT 或 PDF

> **提示：** 对于每月重复的分析（如月度质量报告），可以把配置保存为 YAML 模板，下次用 CLI 一键运行。

### 方式二：YAML 模板 + 命令行（重复性分析）

当你每个月都要对同一份报表做同样的分析时，用模板避免重复配置。

**1. 创建模板文件**（如 `monthly_quality.yaml`）：

```yaml
task: anova
target_col: "不良率"
feature_cols:
  - "料温"
  - "模温"
  - "注射压力"
  - "保压时间"
params:
  alpha: 0.05
output:
  format: [excel, ppt]
```

**2. 运行：**

```bash
smartexcel run monthly_quality.yaml -i 2026年6月质量数据.xlsx -s "注塑车间"
```

**3. 查看所有可用的分析方法：**

```bash
smartexcel list
```

### 方式三：Python 脚本（灵活定制）

```python
import pandas as pd
from smartexcel.core.contracts import AnalysisRequest
from smartexcel.services.orchestrator import orchestrate
from smartexcel.services.reporter import to_ppt

# 1. 读取数据
df = pd.read_excel("质量数据.xlsx")

# 2. 创建分析请求
req = AnalysisRequest(
    task="anova",
    data=df,
    target_col="不良率",
    feature_cols=["料温", "模温", "注射压力", "保压时间"],
    params={"alpha": 0.05}
)

# 3. 运行分析
result = orchestrate(req)

# 4. 查看结论
print(result.summary)   # "显著影响「不良率」的因子: 料温(p=0.003), 模温(p=0.021)"
print(result.tables)    # ANOVA 表、系数表
print(result.figures)   # matplotlib 图表对象

# 5. 导出报告
to_ppt(result, "质量分析报告.pptx")
```

## 如何解读结果？

### 相关性分析

**输出：** 相关系数矩阵 + p 值矩阵 + 一句话结论

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| `r = 0.312，与「不良率」最相关的是「熔体温度」` | 熔体温度每升高 1 个标准差，不良率上升约 0.312 个标准差 | 优先调整熔体温度做验证实验 |
| `r ≈ 0` | 该因子与结果没有线性关系 | 排除该因子，或尝试非线性方法（响应面） |
| `p 值 < 0.05` | 相关性在统计上可信 | 可以据此做决策 |
| `p 值 > 0.05` | 相关性可能是随机波动 | 需要更多数据确认 |

### ANOVA

**输出：** 方差分析表 + 一句话结论

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| `原料类型(p=0.0000) 对冲击强度有显著影响` | 不同材料之间的冲击强度差异不是偶然的 | 如果要提升冲击强度，换材料是有效的杠杆 |
| `未发现显著影响的因子 (α=0.05)` | 在你给的数据中，没有因子能解释 Y 的变化 | 检查数据量是否充足，或者因子范围太窄 |
| `R² = 0.82` | 这个模型能解释 82% 的变差 | 模型拟合很好，可以用于预测 |
| `R² = 0.12` | 模型只能解释 12% 的变差 | 可能有重要的因子没被测量，或者过程噪声大 |

### 假设检验

**输出：** 检验结果表 + 一句话结论

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| `旧工艺 vs 新工艺: 存在显著差异 (p=0.0019)` | 新旧工艺的差异不是偶然的，新工艺确实不同 | 确认差异方向（新工艺更好还是更差），决定是否切换 |
| `未发现显著差异 (p=0.42)` | 数据没有足够证据说明两者不同 | 可能确实没差异，也可能是样本量太小 |

### 回归建模

**输出：** 回归系数表 + R² + 一句话结论

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| `料温 系数=0.06` | 料温每升高 1°C，强度平均增加 0.06 MPa | 用于预测新参数下的质量和设定控制范围 |
| `显著变量: 1/6` | 6 个因子中只有 1 个真正有用 | 可以考虑只保留显著因子，简化控制 |
| `p值 < 0.05` 的变量 | 这个变量的效应是可信的 | 重点关注 |
| `p值 > 0.05` 的变量 | 不能确定这个变量真的有影响 | 收集更多数据或从模型中移除 |

### 决策树

**输出：** 特征重要性排序

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| `关键影响因子: 料温 (重要性=0.387)` | 在所有因子中，料温对结果的解释力最强（占 38.7%） | 优先控制料温，它的波动对质量影响最大 |
| 多个因子重要性相近 | 问题可能是多因子的，没有单一主因 | 用响应面方法同时优化多个参数 |
| 某因子重要性=0 | 决策树认为这个因子完全没用 | 检查是否因为数据中该因子基本不变 |

### 响应面分析

**输出：** 3D 曲面图 + 最优区域

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| 曲面有一个明显的"山头" | 存在最优参数区域 | 将参数设定在"山头"附近 |
| 曲面平坦 | 在测试范围内参数变化对结果影响不大 | 可以放宽该参数的控制范围，降低成本 |
| 曲面有"山谷" | 可能存在鞍点或局部最优 | 扩大搜索范围，或使用网格搜索确认 |

### SPC 控制图

**输出：** X-bar + R 控制图 + 控制限

| 你看到 | 含义 | 怎么做 |
|--------|------|--------|
| 所有点在控制限内，无异常模式 | 过程受控，稳定 | 继续保持，定期监控 |
| 有点超出 UCL/LCL（红虚线外） | 过程失控，存在特殊原因 | 立即排查该时间点发生了什么 |
| 连续 7 个点在中心线同一侧 | "连串规则"——虽然没超线，但过程已经偏移 | 提前干预，检查设备/模具磨损 |
| 点形成上升/下降趋势 | 过程在缓慢漂移 | 检查刀具磨损、温度传感器漂移、原料批次变化 |

### 过程能力 Cp/Cpk

| Cp/Cpk 值 | 判定 | 含义 |
|-----------|------|------|
| Cpk ≥ 1.67 | 优秀 | 过程能力充足，可以考虑放宽抽检频率 |
| 1.33 ≤ Cpk < 1.67 | 合格 | 行业通用标准，正常控制即可 |
| 1.0 ≤ Cpk < 1.33 | 需改进 | 过程能力不足，不良品风险较高 |
| Cpk < 1.0 | 严重不足 | 必须立即改进，当前过程无法满足规格要求 |

## 完整分析方法速查

```bash
$ smartexcel list
  - anomaly_detect      # 异常检测 (IQR / Z-score)
  - anova               # 多因子方差分析
  - correlation         # Pearson 相关性矩阵 + p 值
  - decision_tree       # 决策树特征重要性排序
  - doe_analysis        # DOE 主效应分析
  - grid_search         # 网格搜索最优参数
  - hypothesis_test     # 两样本假设检验 (t 检验 / MWU)
  - multi_objective     # 多目标优化 (期望函数法)
  - process_capability  # 过程能力 Cp/Cpk
  - regression          # 线性回归建模 (OLS)
  - response_surface    # 响应面分析 (3D 曲面图)
  - spc_xbar            # SPC 控制图 (X-bar / R)
  - trend_forecast      # 趋势预测 + 置信区间
  - vif                 # 共线性诊断 (VIF)
```
